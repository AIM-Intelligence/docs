## 세 단계로 시작하기

첫 번째 AIM Monitor 프로젝트를 설정하고 AI 모델 상호작용 추적을 시작하세요.

### 1단계: 조직 및 프로젝트 생성

<AccordionGroup>
  <Accordion title="조직 생성하기" icon="building">
    1. [조직](https://monitor.aim-intelligence.com) 페이지로 이동합니다
    2. "조직 생성" 버튼을 클릭합니다
    3. 조직 이름을 입력합니다
    4. 생성을 확인합니다
  </Accordion>
  <Accordion title="첫 번째 프로젝트 생성하기" icon="folder">
    1. 조직 내에서 "프로젝트 생성" 버튼을 클릭합니다
    2. 프로젝트 이름을 입력합니다
    3. 환경을 선택합니다 (기본값, 프로덕션, 스테이징 등)
    4. 프로젝트 API 키를 복사합니다 - 다음 단계에서 필요합니다

    <Tip>
      API 키를 안전하게 보관하고 버전 관리 시스템에 커밋하지 마세요.
    </Tip>
  </Accordion>
</AccordionGroup>

### 2단계: 첫 번째 트레이스 전송

<AccordionGroup>
  <Accordion title="API 사용하기" icon="code">
    간단한 API 호출로 AIM Monitor에 트레이스를 전송합니다:

    ```bash
    curl -X POST https://api.aim-intelligence.com/v1/traces \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "name": "chat-completion",
        "input": "AI 모니터링이란 무엇인가요?",
        "output": "AI 모니터링은...",
        "metadata": {
          "model": "gpt-4",
          "user_id": "user123"
        }
      }'
    ```

    <Tip>
      `YOUR_API_KEY`를 1단계에서 받은 API 키로 바꾸세요.
    </Tip>
  </Accordion>
</AccordionGroup>

### 3단계: 데이터 확인하기

<Accordion title="대시보드 살펴보기" icon="chart-line">
  1. 프로젝트 대시보드로 이동합니다
  2. [트레이싱](/ko/observability/tracing) 테이블에서 첫 번째 트레이스를 확인합니다
  3. 메트릭, 비용, 성능 데이터를 탐색합니다
  4. 필터를 설정하여 특정 데이터를 분석합니다

  <Tip>
    데이터는 일반적으로 전송 후 몇 초 내에 나타납니다.
  </Tip>
</Accordion>
